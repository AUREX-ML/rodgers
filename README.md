# Rodgers Portfolio Landing Page

A modern GitHub Pages-ready portfolio and personal brand landing page for Rodgers: technologist, environmentalist, and engineer.

## Visual Direction

African energy futurism: solar gold, grid blue, earth tones, deep green interface, glowing system-card aesthetics, and energy-network motion.

## Folder Structure

```text
portfolio/
├── index.html
├── README.md
├── assets/
│   ├── css/style.css
│   ├── js/script.js
│   ├── images/profile-placeholder.svg
│   ├── docs/rodgers-cv.pdf
│   ├── icons/
│   └── fonts/
├── sections/
└── data/projects.json
```

## Replace These Placeholders

1. Replace `assets/images/profile-placeholder.svg` with your real profile photo.
   - Recommended filename: `profile.jpg`
   - Then update the image path inside `index.html`.

2. Add your CV file to:

```text
assets/docs/rodgers-cv.pdf
```

3. Update contact links inside `index.html`:

```html
mailto:your.email@example.com
https://github.com/your-github-username
https://linkedin.com/in/your-linkedin-username
https://x.com/your-handle
```

## Publish on GitHub Pages

1. Push this folder to a GitHub repository.
2. Go to repository **Settings**.
3. Open **Pages**.
4. Select branch: `main`.
5. Select folder: `/root`.
6. Save.

GitHub will generate a public website link for the landing page.
